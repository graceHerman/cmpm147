Technical

The drawGrid function visualizes the grid using the placeTile function, and applies autotiling to create smoother transitions. This function also applies visual effects, such as animated water bubbles.

Overworld

"~" for water

"," for plains

"." for grass

"^" for houses

"\*" for trees

Dungeon

"-" for background/grid

"." for rooms

"=" for hallways

"\*" for chests
